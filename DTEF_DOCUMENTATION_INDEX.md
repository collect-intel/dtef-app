## üìù Document Maintenance

**When to update documentation:**

- **After completing a phase:** Update task status in `DTEF_TASK_PLAN.md`
- **After architecture changes:** Update `PROJECT_CONTEXT.md` and `DTEF_AGENT_ARCHITECTURE.md`
- **After major decisions:** Document in `PROJECT_CONTEXT.md`
- **After adding features:** Update relevant sections in `DTEF_TASK_PLAN.md`

**Document owners:**
- All agents are responsible for keeping documentation up to date
- Update docs as you work, not after
